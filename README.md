# proyecto-python
Trabajo Final Integrador

Este proyecto consiste en un sistema de gestión de turnos hospitalarios
desarrollado en Python y ejecutado por consola.

Funcionalidades

-   Registrar pacientes.
-   Asignar turnos médicos.
-   Registrar la atención de un paciente.
-   Consultar la información de los pacientes.
-   Mostrar estadísticas del sistema.

Tecnologías utilizadas

-   Python 3
-   Programación estructurada
-   Consola

Contenidos aplicados

-   Variables
-   Estructuras condicionales (if, elif, else)
-   Estructuras repetitivas (while y for)
-   Funciones
-   Listas
-   Diccionarios
-   Validaciones
-   Contadores
-   Modularización del código

Menú principal

1.  Registrar paciente
2.  Asignar turno
3.  Registrar atención
4.  Consultar pacientes
5.  Estadísticas
6.  Salir

Validaciones implementadas

-   DNI válido.
-   DNI no repetido.
-   Nombre no vacío.
-   Edad válida.
-   Especialidad existente.
-   Prioridad existente.
-   Horario válido.
-   Turnos ocupados.
-   Pacientes sin turno asignado.
-   Pacientes ya atendidos.

Estructura de los pacientes

Cada paciente se almacena mediante un diccionario con: - DNI - Nombre -
Edad - Especialidad - Turno - Prioridad - Estado de atención

Todos los pacientes se almacenan dentro de una lista, permitiendo
realizar consultas y generar estadísticas.

Ejecución

1.  Abrir el proyecto en Visual Studio Code.
2.  Ejecutar el archivo PhytonTrabajogrupal.py.
3.  Utilizar las opciones del menú principal.
